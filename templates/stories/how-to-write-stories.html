How to write stories on this site?

See this page to write stories on this site.

In this site, you can also write you problems, consequences faced and get a way to achieve a goal.

  1. First create an account
  2. Next click on the Hi, (your name) under the User section.
  3. Click on the option WRITE STORIES.
  4. Select the desired type for which you want to write a post.
Congratulations, you have created a post.
